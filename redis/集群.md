### 为什么最少三个master节点？
因为投票机制  
集群中节点的fail是通过集群中超过半数的节点检测失效时才生效  
例如只有两个节点,挂掉一个,剩下一个投票是不会超过50%的,所以最少要三个节点

### PHP连接集群
```php
$redis = new \RedisCluster(NULL, ['127.0.0.1:30001']);
// 管道技术
$pipe = $redis->multi(\Redis::PIPELINE);
for ($i = 0; $i < 3; $i++) {
    $key = "key::{$i}";
    dump($pipe->set($key, str_pad($i, 2, '0', 0)));
    echo PHP_EOL;
    dump($pipe->get($key));
    echo PHP_EOL;
}
$result = $pipe->exec();
dump($result);
```
